**Gap Types:**
- `Source Missing` — a required log source has not been onboarded to Splunk at all
- `Retention Mismatch` — the source is onboarded but retained for less than the required period
- `Immutable Storage Missing` — the source is onboarded but does not write to a WORM/Object Lock archival tier
- `CIM Non-Compliant` — the source does not map to the required CIM datamodel, breaking correlation search coverage
- `Completeness Gap` — the source is partially onboarded (not all hosts reporting, sampling applied where completeness is required)
