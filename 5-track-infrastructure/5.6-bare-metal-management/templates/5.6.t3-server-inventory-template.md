## Column Definitions

| Column | Format / Values | Required |
|--------|----------------|---------|
| Asset ID | TM-SRV-XXXX (sequential, never reuse) | Yes |
| Hostname | FQDN or short hostname | Yes |
| Make / Model | Vendor + product line + model number | Yes |
| CPU Config | [count]x [CPU model] ([core count]c each) | Yes |
| RAM (GB) | Total GB, ECC type (RDIMM/LRDIMM) | Yes |
| Boot Disk | [count]x [capacity] [type] RAID [level] | Yes |
| Data Disks | [count]x [capacity] [type] RAID [level] or JBOD; dash if none | Yes |
| IPMI/iDRAC IP | Static IPv4; never DHCP | Yes |
| Mgmt VLAN | VLAN ID for management network | Yes |
| Rack Location | DC/Row/Rack/RU position | Yes |
| Install Date | YYYY-MM-DD | Yes |
| Warranty Expiry | YYYY-MM-DD; flag expired | Yes |
| Refresh Target | Quarter/Year (Q1 2026) | Yes |
| Support Contract | Vendor + contract type | Yes |
| Support Tier | NBD / 4hr / 24x7x4 / Mission Critical | Yes |
| OS / Image | OS name and version | Yes |
| Role / Cluster | Workload description or cluster membership | Yes |
| Notes | Any special config, flags, or exceptions | Optional |
