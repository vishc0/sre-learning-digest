- **Site/Datacenter**: Physical location (DC name or colo facility)

## Section 7: License Risk Flags

Review this checklist after completing the inventory. Each checked item is a risk to address before renewal.

**License Risk Checklist**:
- [ ] Any ESXi hosts on Evaluation licenses (they expire — convert before renewal)
- [ ] Any ESXi hosts on versions earlier than 7.0 (End of General Support — security risk)
- [ ] vCenter instances without HA (single point of management failure)
- [ ] Auto-renewal clause present with <90 days to notice deadline
- [ ] NSX deployed but not covered by current license (entitlement gap)
- [ ] vSAN deployed without vSAN Enterprise on clusters using advanced features (encryption, stretched cluster)
- [ ] License keys applied that expired >30 days ago (runs in grace period — check)
- [ ] Core count has grown since last renewal (new servers added = unreported billing exposure)
