## 2. Change Classification

Select all that apply. Each classification activates specific risk items in Section 4.

### 2.1 BGP Policy Change
- [ ] Routing policy modification (route-map, prefix-list, community filter)
- [ ] Prefix advertisement change (adding or withdrawing prefixes from BGP)
- [ ] AS path manipulation (prepending, local preference adjustment)
- [ ] Peer relationship change (new peer, peer removal, peer group modification)
- [ ] Route redistribution change (OSPF into BGP or reverse)

### 2.2 Firewall Rule Change
- [ ] New rule addition (permitting previously blocked traffic)
- [ ] Rule modification (changing src/dst/port on existing rule)
- [ ] Rule deletion (removing a rule — highest blast radius)
- [ ] Rule reorder (order matters in stateful firewalls — reordering can shadow rules)
- [ ] ACL modification on router/switch interface

### 2.3 DNS Record / TTL Change
- [ ] A or AAAA record change (IP address change)
- [ ] CNAME change (alias target change)
- [ ] TTL reduction (pre-failover TTL drain — must be done ≥ current TTL before cut)
- [ ] TTL increase (post-failover normalization)
- [ ] NS delegation change (highest risk DNS change)
- [ ] Zone transfer or authoritative server change

### 2.4 Load Balancer Configuration Change
- [ ] Backend pool member add/remove
- [ ] Health check modification (threshold, path, interval)
- [ ] SSL certificate rotation
- [ ] Listener rule change (routing rules, host-based or path-based)
- [ ] Connection timeout or keep-alive setting change
- [ ] Upstream TLS termination change

### 2.5 Network Segmentation Change
- [ ] VLAN add/modify/remove
- [ ] Security group rule change (AWS/GCP/Azure)
- [ ] Subnet CIDR change (extremely high risk — requires coordination)
- [ ] VPC peering or transit gateway change
- [ ] Network ACL change

### 2.6 Physical / Cabling Change
- [ ] Physical cable installation or re-patch
- [ ] Optic replacement or module swap
- [ ] Port configuration (speed, duplex, LACP)
- [ ] Maintenance on live circuit (fiber cleaning, connector replacement)
