**Days to 80% formula**: `(Total_Capacity × 0.80 - Used_Capacity) / 30-day_Growth × 30`  
**Days to Full formula**: `(Total_Capacity - Used_Capacity) / 30-day_Growth × 30`

**Status key**: OK = >120 days to 80% | WARN = 60–120 days | CRIT = <60 days | EMERG = <30 days

**Storage tier decision criteria** (apply before provisioning):
- Latency requirement < 1ms → Block (SAN/EBS/Local NVMe)
- Shared access required → NAS (NFS/SMB) or EFS
- Retention > 90 days with infrequent access → Object storage with lifecycle policy
- Compliance/WORM requirement → Object with Object Lock
- Kubernetes RWO → EBS or local path
- Kubernetes RWX → EFS or CephFS
