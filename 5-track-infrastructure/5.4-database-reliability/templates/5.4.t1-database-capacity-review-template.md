# Database Quarterly Capacity Review Template

> **How to use this template**: Complete before the quarterly review meeting. Pull all metric values from your monitoring system (Prometheus/Grafana, Splunk, Datadog) for the period covered. Projections should be based on observed trend data — not guesses. Present the Recommendations table (Section 8) and Action Items (Section 10) to the group for prioritization decisions. File the completed document in your team's Confluence or shared drive and link from the database runbook.
