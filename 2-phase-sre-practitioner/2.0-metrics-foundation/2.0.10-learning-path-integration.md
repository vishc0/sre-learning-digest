# Textbook → Learning Path Integration Guide

The Textbook teaches a proprietary reliability measurement framework. Each chapter's metrics should be studied ALONGSIDE the domain framework where those metrics are applied operationally.

## When to Read the Textbook

**Don't start the Textbook in Phase 1.** The Textbook assumes you understand incidents, SLOs, and error budgets. Start the Textbook at Phase 2, chapter by chapter, alongside the domain framework where each metric is applied.

## Textbook Study Discipline

Don't read the Textbook for its own sake. For each chapter:
1. Read the chapter
2. Open the corresponding domain framework chapter
3. Find where the metric shows up in operational practice
4. Practice calculating the metric against a real or synthetic example
5. Return to ch09 for daily application

The Textbook metrics become useful ONLY when you can calculate them from real incident/SLO/change data. Without that practice, they are vocabulary without meaning.
