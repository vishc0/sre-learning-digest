# Template — Weekly Executive Status Update

> **Usage notes:** Fill in all bracketed fields. Delete all guidance text (in italics or brackets) before sending. Maximum length: one page when printed or rendered. Send in email body — not as an attachment.

---

## Platform Engineering Status — Week of [DATE]

**Author:** [Your Name] | **Distribution:** [VP Name], [CTO if applicable]
**Status:** 🟢 GREEN / 🟡 AMBER / 🔴 RED

> *[Choose one status for the overall week. GREEN = no executive action needed. AMBER = something elevated, awareness required. RED = executive action or attention needed now. RED should not appear here for the first time — executive should already know via real-time communication.]*

---

### Headline

> *[One sentence. The single most important thing the executive needs to know this week. If everything is normal: "Platform performed within SLA targets across all services this week with no material incidents." If something happened: "A 47-minute payment service outage on Thursday impacted 18,000 customers; root cause identified and fix deployed — post-incident summary below." Write the sentence you would say if you had 10 seconds in an elevator.]*

**[Your one-sentence headline here]**
