# Cookbook → Learning Path Integration Guide

The Cookbook provides conceptual vocabulary and interview preparation. For each topic, this guide tells you which learning phase it belongs to, what domain framework expands on it operationally, and what to read before and after.

## Reading Before the Cookbook (Phase 1 Prerequisites)
Before starting the Cookbook, complete:
- weeks/week-01-linux-basics/ — you need command line comfort
- weeks/week-03-kubernetes/ — Cookbook references K8s throughout
- weeks/week-05-monitoring/ — Cookbook observability topics assume basic query familiarity
